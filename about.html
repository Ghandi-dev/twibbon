About Twibbon

Twibbon Maker

Script untuk membuat twibbon sederhana tanpa edit.

Cara Penggunaan?

  1. Download scriptnya dan Upload script ini di cpanel.

  2. Ganti file twibbon.png untuk menggantikan twibbon.

  3. Untuk mengubah ikon, harap sesuaikan dengan file .ico Anda di HTML.

  4. Selesai.

Tolong jangan hapus apapun selain icon dan twibbon untuk mendukung kami! Jangan jual skrip ini!.
